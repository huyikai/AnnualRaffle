# 用户照片目录

此目录用于存放用户照片文件。

## 文件命名规则

照片文件必须以用户的 `key` 值命名，格式为：`{key}.jpg` 或 `{key}.png`

例如：
- 如果用户的 key 是 `1001`，则照片文件名应为 `1001.jpg`
- 如果用户的 key 是 `2200015`，则照片文件名应为 `2200015.jpg`

## 照片要求

- **格式**：支持 `.jpg` 和 `.png` 格式
- **大小**：单张照片不能超过 `150kb`，建议 `20-50kb`
- **尺寸**：建议尺寸为 `160*160px`（正方形）
- **质量**：建议使用压缩后的照片以提升加载速度

## 使用说明

1. 将用户照片按照命名规则放入此目录
2. 照片文件名必须与 `src/assets/user.ts` 中用户的 `key` 值完全一致
3. 如果某个用户没有照片，程序会自动使用默认显示方式（仅显示姓名或号码）

## 注意事项

- 此目录中的真实照片文件不会被提交到 Git 仓库（已在 .gitignore 中配置）
- 如需使用示例照片，请参考项目根目录的 README.md 中的配置说明
- 建议在导入照片前先压缩照片大小，以提升程序运行性能

## 批量导入

如果照片较多，建议：
1. 使用图片压缩工具批量压缩照片
2. 确保所有照片文件名与用户 key 值匹配
3. 一次性将所有照片放入此目录
